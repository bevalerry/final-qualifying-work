Веб-приложение образовательной платформы

Описание: Платформа для автоматической генерации адаптивных учебных материалов на основе лекционного контента с использованием микросервисной архитектуры и нейросетевого модуля.

Технологии: Java, Spring Boot, PostgreSQL, React, Redux, TypeScript, REST, нейросеть (GigaChat API)

Структура репозитория:

/frontend — клиентская часть
/backend  — серверная часть и микросервисы
/docs     — диаграммы, документация


Цель: Создание платформы для автоматической генерации учебных материалов.
